Contribute

Join the open‑source project, improve your skills and help build Zero Day Exploit. All contributions are welcome — code, 3D models, media and community support.

Who We’re Looking For

  • ✔ Experienced programmers (Godot, C#, Python)
  • ✔ Good 3D modelers (hard‑surface preferred)
  • ✔ UI/UX designers
  • ✔ Music & sound creators
  • ✔ Social media helpers

Contribution Rules

  • ✔ Code must be clean & readable
  • ✔ PRs must explain what changed
  • ✔ Features must fit the game’s theme
  • ✔ No copyrighted assets
  • ✔ Ownership of the project remains with the creator

Good vs Bad Pull Requests

Good PR Bad PR
Clear description, screenshots, clean code No explanation, random files, messy changes
Fixes one problem Changes 20 files without reason
Matches project style Writes chaos code
